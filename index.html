Weibo spider with face detection

A web scraper in Python to download tweets, photos and user information from Chinese social website Weibo for data mining use. It also calls a face detection API to automatically recognize portraits from all photos, and store the biological details of faces for further research.

More details please read README on GitHub

Remote Procedure Call(RPC)

Implemented a crude version of Remote Procedure Call(RPC) in C and C++. This system including several clients who request the servers to do their jobs, a binder who tells clients which server can do that job and maintains a servers schedule to balance workloads, and several servers who call the disred server procedure with client's parameters.

Intrusion Detection System(IDS)

Implemented a simple Intrusion Detection System using Python that runs on the router of a local network to monitor network traffic for known attacks based on provided signatures. Detections including Anomaly detection, Spoofed packets, Unauthorized servers,Sinkhole lookups, ARP spoofing, IIS worms, and NTP reflection DDoS.
